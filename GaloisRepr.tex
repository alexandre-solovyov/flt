\section{Представления Галуа mod \textit{p}}

Пусть $E$ обозначает фиксированную эллиптическую кривую над $\Q$.
Для данного простого $p$ обозначим $E[p]$ подгруппу
$E(\overline{Q})$, состоящую из точек порядка $p$,
лежащих в алгебраическом замыкании $\overline{\Q}$.

Тогда $E[p] \sim \F_p^2$, рассматриваемому как 2-мерное
векторное пространство (модуль) над $\F_p$.

Группа Галуа $\Galq$ действует на точках кривой,
отображая кривую в себя: в самом деле,
если кривая задается уравнением $f(x,y)=\sum a_{ij} x^i y^j = 0$,
коэффициенты полинома из основного поля ($\Q$),
$\sigma \in \Galq$, то
$$
  0 = \sigma(f(a,b)) 
    = \sigma(\sum a_{ij} a^i b^j)
	= \sum a_{ij} (\sigma a)^i (\sigma b)^j
	= f( \sigma a, \sigma b )
$$

Действие группы Галуа на точках сохраняет групповой закон на кривой.
Доказательство: координаты суммы точек на эллиптической кривой
выражаются как рациональные функции координат точек, т.е.
$$
  X_{P+Q} = R( X_P, Y_P, X_Q, Y_Q )
$$
$$
  \sigma X_{P+Q} = \sigma R( X_P, Y_P, X_Q, Y_Q )
  = R( \sigma X_P, \sigma Y_P, \sigma X_Q, \sigma Y_Q )
$$

Аналогично для $Y_{P+Q}$. В итоге:
$$
  \sigma (P+Q) = \sigma P + \sigma Q 
$$

В частности, под действием группы Галуа точки порядка $p$
переходят в точки порядка $p$.


Таким образом, действие группы Галуа на точках можно описать
через автоморфизмы группы $E[p]$, т.е. автоморфизмы $\F_p^2$.
Автоморфизмы векторного пространства описываются полной линейной
группой $\GL$, в  нашем случае $\GL_2(\F_p)$.

Таким образом, для кривой $E$ и простого числа $p$ определено
отображение:
$$
  \gr : \Galq \to \GL_2(F_p)
$$
(однозначно с точностью до изоморфизма).

Это отображение $\gr$ кодирует арифметику кривой $E$, что видно
из двух следующих ключевых свойств:

(a) пусть $\omega_p : \Galq \to \F_p^{\times}$ -- характер,
задающий действие группы Галуа на корни $p$-й степени из единицы:
$\mu_n$; подробнее, перестановка из группы Галуа переводит 
корень из единицы в другой корень из единицы: $\zeta \to \zeta^m$.

Тогда $\det(\gr) = \omega_p$, это следует из ''пар Вейля'':
$E[p] \times E[p] \to \mu_p$, совместимым с действием группы Галуа.
