Теорема Морделла-Вейля утверждает, что для любой эллиптической
кривой $E$ над числовым полем $K$, группа точек кривой с
координатами из поля: $E(K)$ -- конечно порождена.

Эта теорема была доказана Морделлом в 1922 году для $K=\Q$ и
обобщена для произвольного числового поля Вейлем в 1928 году.

\textbf{Алгебраическое числовое поле} (или просто 
\textbf{числовое поле}) — это конечное (а следовательно --
алгебраическое) расширение поля рациональных чисел $\Q$. 

Для любой абелевой группы $M$ если $M$ -- конечно порождена,
то $M/nM$ -- конечна для всех $n>1$.
Обратное в общем случае неверно, например, $(\Q,+)$ имеет свойство
$Q = nQ$, но элементы любой конечно порожденной подгруппы $\Q$ имеют
ограниченный знаменатель, значит $\Q$ не является конечно
порожденным.

Поэтому для точек $E(K)$ дополнительно определяется функция высоты: 
$h: E(K) \to \R_{+}$ со свойствами:

1) $h$ совместима с операцией сложения точек: $h(P+Q) = h(P)+h(Q)$;

2) для любой константы $B$ множество точек $P$ таких, что $h(P)<B$
конечно;

3) высота увеличивается для кратных точек: $h(nP) \ge n h(P)$, $n>1$.

Тогда доказательство теоремы проводится так.
Пусть например, $n=2$, т.е. мы работаем с факторгруппой
$E(K)/2E(K)$. Она конечна, и мы обозначим через $P_1, ..., P_s$
представителей каждого смежного класса.
Выберем любую фиксированную константу $C$ и обозначим точки
с высотой меньше $C$: $Q_1, ..., Q_t$ (их конечное число по
свойству 2).

Выберем любую точку $Q \in E(K)$ и покажем, что она выражается
конечной линейной комбинацией точек $P_1, ..., P_s, Q_1, ..., Q_t$.

Эта точка представляется как $Q = P_i + 2Q'$ для некоторого $i$
и некоторой точки $Q' \in E(K)$.
Если высота $Q'$ меньше $C$, то она уже находится среди
$Q_1, ..., Q_t$.
Если высота $Q'$ не меньше $C$, то к ней применим тот же аргумент:
$$
  Q' = P_j + 2Q''
$$
$$
  Q = P_i + 2(P_j + 2Q'')
$$

Высота $h(Q') = h(P_j) + 2h(Q'')$, отсюда $h(Q'') < h(Q')/2$.
Таким образом мы получили точку с высотой в два раза меньше.
Продолжая этот процесс, мы получим в итоге точку с высотой
меньше $C$ и процесс остановится, и для точки $Q$ 
получится конечная линейная комбинация точек
$P_1, ..., P_s, Q_1, ... Q_t$.

Этот способ называется доказательство спуском, широко
использовавшимся Ферма.
Однако, в некотором смысле этот способ восходит еще к древним 
грекам, который так доказывали иррациональность квадратного корня
из 2.

Таким образом, для окончания доказательства надо установить
конечность группы $E(K)/nE(K)$ и определить функцию высоты
на точках эллиптической кривой.
